Cara Menjalankan, 
Gunakan aplikasi XAMPP Control Panel v3.3.0 untuk menyalakan module MySQL. 
Buka folder proyek “Basis Data Tubes” menggunakan Visual Studio Code.
Pastikan interpreter Python yang digunakan adalah virtual environment (venv) yang terdapat pada proyek.
Jalankan file utama aplikasi (app.py) menggunakan tombol Run Python File pada Visual Studio Code.
Tunggu hingga server Flask berhasil berjalan dan menampilkan alamat localhost, misalnya: http://127.0.0.1:5000 
Buka browser, kemudian akses alamat tersebut untuk menampilkan dashboard.
Dashboard akan menampilkan fitur pengelolaan data mahasiswa, mata kuliah, nilai, transkrip akademik, serta visualisasi data.
